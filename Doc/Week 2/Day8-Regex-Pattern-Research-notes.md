Today I'm going to learn about regex.

Regular expressions, or regex, are a pattern language used to search, match, or replace text based on a specific pattern. Regex is only powerful if it provides a pattern for searching, matching, or replacing. Regex works by providing a specific pattern for each specific condition. For example, if a regex contains a period (.) like a.c, then any combination like abc, a-c, azc is considered a regex pattern. The exception is the letter ac, which cannot be used as a regex pattern. So, what characters support regex patterns? There are several, namely the dot (.) is used for one character, there is also (\d) which means the number only has (0-9) which can only accommodate numbers and has a combination such as (\d\d) for two numbers and so on, there is also (+) which indicates one or more and can be applied in all regex patterns, there is also (\*) which indicates zero or more and can be applied in all regex patterns, there is also (?) which means optional, can be a combination or no combination, for example color can also be color and color, there is also [] which means character class, for example [a-z] which means it can only accept a combination of letters a-z which are lowercase letters, there is also () which means grouping, for example (abc) then the acceptable combination of the regex pattern is abc only in the grouping. What conditions make regex suitable for use? When the sentence or word being searched has a pattern that can be used for filtering, searching, and matching. The regex pattern is fast in assessing patterns and is lightweight because it does not require an AI model. This pattern is suitable for handling data that has the same pattern. What conditions make regex unsuitable for use? In terms of concepts, since regex patterns only recognize patterns without understanding the concept, for example, for a name, an AI model would definitely be used to verify the name rather than using regex patterns that must have conditions that must be met and do not recognize the concept. Python has a built-in library called `re` to use regular expression patterns in Python. Python regular expressions are very useful for data validation, data searching, and data matching. To call the library, simply use `import re` to enable regular expression patterns in Python.
